# 副業コンパスAI — プロジェクト概要

## サービスコンセプト

「副業コンパスAI」は副業診断サービスではありません。

10〜15問の選択式質問に答えるだけで、ユーザー本人も気づいていない行動特性・強み・行動パターンをAIが言語化し、
「向いている働き方タイプ」を特定したうえで、具体的な副業候補と最初の一歩を提示するサービスです。

「自分には何もない」と思っていた人が「実はこんな特徴があった」と気づき、
その日のうちに最初の一歩を踏み出せることを最終目的とします。

---

## ターゲットユーザー

### メイン（最優先）
- 30〜50代の会社員
- 副業を始めたいが何から始めればいいかわからない人
- AI初心者
- 営業が苦手な人

### サブ
- 主婦・主夫（スキマ時間に稼ぎたい）
- 独立・フリーランスを考えているが迷っている人

---

## コアバリュー

| 価値 | 説明 |
|------|------|
| 言語化 | 自分でも気づいていない特徴・強みを言葉にする |
| 行動 | 診断で終わらず「今日やること」まで提示する |
| 継続 | 7日間ロードマップで最初の1週間を伴走する |
| 安心 | 営業不要・在宅・初心者でも始められる副業のみ対象 |

---

## 主な機能（MVP）

| 機能 | 説明 |
|------|------|
| 行動特性診断 | 10〜15問の選択式質問でユーザープロファイルを生成 |
| 強み言語化 | タグプロファイルから行動特性・強みを文章で出力 |
| 働き方タイプ判定 | 5タイプから最も合うタイプを判定・説明 |
| 副業マッチング | 働き方タイプ×行動特性タグで副業ランキングを生成 |
| 今日やること | 副業タイプ別に具体的なアクションを1つ提示 |
| 7日間ロードマップ | 最初の1週間の行動計画をデイリーで提示 |

---

## 診断結果の構成

診断結果は以下の順で表示する。副業名を当てることより、「自分への気づき」を先に届けることを優先する。

1. あなたの行動特性（3〜5項目）
2. 無意識にできていること（強みの言語化）
3. 向いている働き方タイプ
4. 向いていない働き方タイプと理由
5. おすすめ副業ランキング（上位3〜5件・理由付き）
6. おすすめしない副業（2〜3件・理由付き）
7. 今日やること（具体的アクション1つ）
8. 7日間ロードマップ

Webライティング / QAテスター / データ入力 / LP制作 / ホームページ制作 / ノーコード制作 / AI活用代行 / MEO運用代行

## ビジネスフェーズ

| フェーズ | 内容 | 目的 |
|---------|------|------|
| Phase 1 (MVP) | ルールベース診断・完全無料・ログイン不要 | 価値検証 |
| Phase 2 | Claude API連携・詳細診断・30日ロードマップ | 精度向上 |

| Phase 4 | AIチャット相談・学習ロードマップ・進捗管理 | エンゲージメント |

## MVPの検証ゴール

- 診断完了率 70%以上（離脱を起こさないか）
- 結果への納得感（「参考になった」フィードバック 70%以上）
- 「今日やること」が実際の行動につながるか（定性フィードバック）

## 技術スタック

- デプロイ: Vercel
- 将来: Anthropic Claude API / Supabase（認証・DB）

## サービス運営方針

このサービスは、副業を紹介することが目的ではありません。

初心者が「何をすればいいか分からない」という状態から、自分に合った最初の一歩を見つけ、小さく行動できるよう支援することを目的とします。

---

## 外部サービスへのリンク方針

クラウドワークスやココナラなどの外部サービスへのリンクは、収益化を目的として配置するものではありません。利用者が次の行動を取りやすくするために配置します。

診断結果やランキングは、広告報酬ではなく、診断ロジックのみで決定します。

外部リンクがアフィリエイトリンクとなる場合は、広告であることを明記します。

---

## 収益化方針

収益化は、ユーザー体験を損なわない範囲で行います。

収益を優先して診断結果や導線を変更することは禁止します。

サービスの価値を最優先とし、その結果として自然に収益が発生する形を目指します。

---

## 将来構想

現在は「初心者 → 診断 → 最初の一歩」までを対象とします。

将来的には、副業継続 → 事業化 → 地域支援（商工会・よろず支援拠点等）まで案内できるサービスへ発展させる構想があります。

ただし、この構想は現時点では実装対象ではありません。MVPでは初心者の最初の一歩に集中してください。
