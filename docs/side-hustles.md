# 副業データ一覧

## データ構造

```typescript
interface SideHustle {
  id: string;
  name: string;
  category: "skill" | "content";
  primary_types: WorkStyleTypeId[];   // 相性の良い働き方タイプ
  match_tags: string[];               // マッチングスコアに使うタグ
  aversion_excludes: string[];        // このタグがあるとスコアを下げる
  difficulty: 1 | 2 | 3 | 4 | 5;    // 難易度（1:超簡単 〜 5:上級）
  income_3months: string;             // 3ヶ月後の月収目安
  income_6months: string;             // 6ヶ月後の月収目安
  first_income_days: number;          // 初収入までの目安（日数）
  description: string;               // 一言説明
  why_good: string;                  // 向いている人への説明
  today_action: string;              // 今日やること（10〜30分で完了）
  week_roadmap: string[];            // 7日間ロードマップ（Day1〜Day7）
}
```
